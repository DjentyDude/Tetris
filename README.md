Tetris by DjentyDude

HOW TO RUN: No Python needed, . Just open the dist folder, then Tetris, and run the executable. Have fun!

About: A fully functional Tetris clone built in Python using the PyGame library. Includes all 7 standard tetrominoes, collision detection, line clearing, and color-coded pieces.

Controls: <- -> to move, Up arrow to rotate, Down to drop faster.

Version 1.0--Initial Release, Core Gameplay

Roadmap as follows: Score tracking, Increasing speed per level, Improved UI and styling, Slowing down the first level, Allowing Infinite replays.

Built with Python and PyGame, my first graphical project. More versions coming.
